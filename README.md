# Stretty Music

**Tagline:** Depressions Running Deep  
**Label:** HTG - Hex The Government

A production-ready one-page music website for **Stretty Music**, built with pure **HTML, CSS, and vanilla JavaScript**.

The site is designed for a dark emo alt rap release profile with a terminal / hacker-inspired visual style, floating side navigation, scroll-based storytelling, and interactive UI effects.

---

## Overview

This project is a single-page scrolling music site built for:

- artist branding
- release promotion
- out-now streaming links
- upcoming drops
- merch promotion
- tour announcements
- image gallery / visual rollout
- contact and booking enquiries

The design uses a black and terminal-green palette with a sharp gothic / code-inspired interface.

---

## Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

No frameworks. No build tools. No package installs. No bloated dependency graveyard.

## How to Use

1. Download or clone the repository.
2. Keep `index.html`, `style.css`, and `script.js` in the same folder.
3. Open `index.html` in a browser to preview locally.
4. Upload all files to your hosting platform.

## Deployment

This site can be deployed directly to:

- **Cloudflare Pages**
- **GitHub Pages**
- **Netlify**
- **Vercel**
- any standard web host that serves static files

### Basic upload requirement

Upload these files together:

- `index.html`
- `style.css`
- `script.js`

`index.html` is the entry file.

---

## Customisation

### Change artist branding
Edit in `index.html`:

- site title
- hero copy
- section headings
- release names
- platform links
- social links
- contact email
- tour dates
- merch items

### Change colors
Edit CSS variables near the top of `style.css`.

Main palette currently uses:

- `#0a0a0a` background
- `#00ff88` accent
- `#003322` secondary green
- `#1a1a1a` dark surface

### Change images
Replace placeholder images in `index.html` with:

- release covers
- merch photos
- artist visuals
- gallery images

### Change hero video
Replace the external video source inside the hero `<video>` element in `index.html`.
